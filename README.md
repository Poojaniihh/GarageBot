# 🏎️ AutoGarage Chatbot

A simple AI-powered chatbot that allows users to check **vehicle status** by license number. The chatbot uses a **database-first approach** to fetch accurate vehicle data and **OpenAI LLM** to format responses in a friendly, human-readable way.

---

## 📌 Features

- Fetch vehicle details by license number  
- Show repair status, overall progress, and owner info  
- Friendly AI responses using OpenAI  
- Supports Sri Lankan license formats like `ABC-1234` or `CA-1010`  

---

## 🛠️ Tech Stack

| Layer        | Technology |
|-------------|------------|
| Backend     | Python 3.11, FastAPI |
| Database    | MySQL (XAMPP / phpMyAdmin) |
| AI          | OpenAI API (LLM for response formatting) |
| Environment | python-dotenv for config management |
| HTTP Server | uvicorn |
| SQL Connector | mysql-connector-python |
